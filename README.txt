1.1 Кратко что за проект

Реализована функция АПВ с контролем синхронизма по стандарту МЭК 61850. 
Для демонстрации работы АПВ необходимо было дополнительно реализивать алгоритм релейной защиты. 
Для её реализации была выбрана функция токовой направленной защиты ввиду относительной простоты алгоритма. 
В качестве входных данных использовались пакеты по SV-протоколу, которые проходили предварительную обработку для получения действующих значений токов.

1.3 Инструкция по запуску опытов

1.3.1 Необходимо определить наименование сетевого адаптора. При запуске программы доступные адапторы вводятся в консоль. 
Необходимо выбрать тот, который и выведен в консоль, и доступен в программе playcap.
1.3.2 Необходимо скопровать description адаптора из консоли в lsvs.setNicName("...сюда...") в классе MainX
1.3.3 Запуск статического метода main класса MainX (запустится процесс захвата пакетов)
1.3.4 Открыть программу playcap
1.3.5 Во вкладке "Files" открыть требуемый .pcap файл (sv-пакеты)
1.3.6 Выбрать указанный адаптор
1.3.7 Нажать кнопку "Запуск" (запустится процесс воспроизведения пакетов)
1.3.8 Наблюдать на осцилограммах полученный результат
